This is a ready-to-run Vite + React + Tailwind project (dark neon theme).
## Quick start
1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Build: `npm run build`

Notes:
- Replace `public/avatar.svg` with your headshot (or keep the SVG placeholder).
- Update content in `src/App.jsx` (constants at the top) to change text/links.
- To deploy to GitHub Pages, create a repo named `sandeepiliger.github.io` and push this project; or use Vercel/Netlify.
